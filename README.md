# Wilpower Institute - Eduverse

![Eduverse](https://img.shields.io/badge/Eduverse-Educational%20Platform-teal)
![License](https://img.shields.io/badge/license-MIT-green)

## Overview

A modern educational resource platform where teachers and pupils can purchase quality learning materials using **Airtel Money** and **MTN Mobile Money** via Pesapal.

**Live Backend:** https://eduverse-production-5e0b.up.railway.app

---

## Features

- ✅ Admin & Teacher portals
- ✅ Secure mobile money payments (Pesapal)
- ✅ Wilpower Institute branding
- ✅ Firebase ready (for future storage)
- ✅ Clean, responsive design

---

## Quick Start (Frontend)

1. Download the files (`index.html`, `config.js`, `payment.js`)
2. Place `wilpower-logo.png` in the same folder
3. Open `index.html` in your browser (or host on GitHub Pages)

---

## Backend (Railway)

- Deployed on Railway
- URL: `https://eduverse-production-5e0b.up.railway.app`
- Port: `5000`
- Payment endpoint: `/api/payment/initiate`

---

## Payment Testing

1. Click **"Pay with Airtel / MTN"**
2. Enter your name and email
3. You should be redirected to Pesapal sandbox for testing

**Note:** Currently using **Pesapal Sandbox**. Real money will not be deducted.
